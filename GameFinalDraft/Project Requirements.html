Superman Target Practice Game

Abstract: We have created a target shooting game in which a playable character fires a limitted amount of laserbeams at a wave of enemies. The goal is to hit all enemies without missing and shots.

Functional Requirements

  • Menu(Two Buttons: One to begin game, and one to exit the program)
  • Definite end to game
  • Clear objective
  • Enemies to target
  • Calculated stats at the end of the game

Nonfunctional Requirements

  • Runs smoothly on most machines
  • Developed in Java
  • Easily maintainable with changes to Java
  • Easily customizable
